网络安全相关法律法规

中华人民共和国网络安全法

2017年6月1日施行

一、立法目的

为了保障网络安全，维护网络空间主权和国家安全、社会公共利益，保护公民、法人和其他组织的合法权益，促进经济社会信息化健康发展，制定本法。

二、内容

  1. 网络安全战略与促进：国家制定并不断完善网络安全战略，明确保障网络安全的基本要求和主要目标；支持网络安全技术研发和产业发展，推进网络安全社会化服务体系建设。
  2. 网络运行安全：实行网络安全等级保护制度，要求网络运营者履行安全保护义务；关键信息基础设施运营者还需履行更严格的安全保护义务，包括安全评估、数据本地化等。
  3. 网络信息安全：网络运营者收集、使用个人信息应遵循合法、正当、必要原则，公开收集规则，明示收集目的，并经被收集者同意；禁止非法出售或向他人提供个人信息。
  4. 监测预警与应急处置：国家建立网络安全监测预警和信息通报制度，统筹协调网络安全事件的应急处置工作；网络运营者应制定应急预案，及时处置安全事件。
  5. 跨境数据流动规则：关键信息基础设施运营者在中华人民共和国境内运营中收集和产生的个人信息和重要数据应当在境内存储；因业务需要，确需向境外提供的，应进行安全评估。
  6. 法律责任：对网络运营者、网络产品或服务提供者等主体的违法行为设定了罚款、暂停相关业务、吊销许可证等处罚措施；构成犯罪的，依法追究刑事责任。

三、重要意义

《网络安全法》是我国网络空间法治建设的重要里程碑，是依法治网、化解网络风险的法律重器。它不仅明确了网络空间各主体的权利义务，也为网络安全工作提供了明确的法律依据和保障。

中华人民共和国数据安全法

2021年9月1日施行

一、立法目的

为了规范数据处理活动，保障数据安全，促进数据开发利用，保护个人、组织的合法权益，维护国家主权、安全和发展利益，制定本法。

二、内容

  1. 数据分类分级保护制度：国家根据数据在经济社会发展中的重要程度，以及一旦遭到篡改、破坏、泄露或者非法获取、非法利用，对国家安全、公共利益或者个人、组织合法权益造成的危害程度，对数据实行分类分级保护。
  2. 数据安全管理体制：明确中央国家安全领导机构负责数据安全工作的决策和议事协调；各地区、各部门对本地区、本部门工作中涉及的数据安全负主体责任。
  3. 数据处理者义务：数据处理者应建立健全数据安全管理制度，组织开展数据安全教育培训；开展数据处理活动应进行数据安全影响评估，并向有关主管部门报送评估报告。
  4. 数据安全审查：对影响或者可能影响国家安全的数据处理活动，以及重要数据的出境安全管理，依法进行国家安全审查。
  5. 政务数据开放：国家推进政务数据开放利用，县级以上人民政府应建立健全数据开放协调机制，明确政务数据开放清单，促进政务数据的共享和开发利用。
  6. 法律责任：对违反本法规定的行为，设定了警告、罚款、暂停相关业务、吊销相关许可证等处罚措施；构成犯罪的，依法追究刑事责任。

三、重要意义

《数据安全法》是数据领域的基础性法律，也是国家安全领域的一部重要法律。它与《网络安全法》《个人信息保护法》共同构建了我国网络空间治理的法律框架，为数据安全提供了全面的法律保障。

中华人民共和国个人信息保护法

2021年11月1日施行

一、立法目的

为了保护个人信息权益，规范个人信息处理活动，促进个人信息合理利用，根据宪法，制定本法。

二、内容

  1. 个人信息处理基本原则：处理个人信息应当遵循合法、正当、必要和诚信原则，不得通过误导、欺诈、胁迫等方式处理个人信息；应具有明确、合理的目的，并应当与处理目的直接相关。
  2. 个人信息处理者义务：个人信息处理者应制定内部管理制度和操作规程，对个人信息实行分类管理；采取相应的技术措施和其他必要措施，保障个人信息的安全。
  3. 个人在个人信息处理活动中的权利：个人对其个人信息享有查阅、复制、更正、删除等权利；发现个人信息处理者违反法律、行政法规规定或者双方的约定处理其个人信息的，有权请求个人信息处理者及时采取删除等必要措施。
  4. 敏感个人信息的特殊保护：处理敏感个人信息应当取得个人的单独同意；法律、行政法规规定处理敏感个人信息应当取得书面同意的，从其规定；处理敏感个人信息还应当向个人告知处理敏感个人信息的必要性以及对个人权益的影响。
  5. 个人信息跨境提供规则：关键信息基础设施运营者和处理个人信息达到国家网信部门规定数量的个人信息处理者，应当将在中华人民共和国境内收集和产生的个人信息存储在境内；确需向境外提供的，应通过国家网信部门组织的安全评估。
  6. 法律责任：对违反本法规定处理个人信息的行为，设定了高额罚款、暂停相关业务、吊销相关许可证等处罚措施；对直接负责的主管人员和其他直接责任人员可以处一万元以上十万元以下罚款。

三、重要意义

《个人信息保护法》聚焦个人信息保护领域的突出问题和人民群众的重大关切，建立健全了科学、系统的个人信息保护制度体系，与民法典、网络安全法、数据安全法等法律共同构建了网络空间治理的法律框架。

关键信息基础设施安全保护条例

2021年9月1日施行

一、立法目的

为了保障关键信息基础设施安全，维护网络安全，根据《中华人民共和国网络安全法》，制定本条例。

二、内容

  1. 关键信息基础设施范围：本条例所称关键信息基础设施，是指公共通信和信息服务、能源、交通、水利、金融、公共服务、电子政务、国防科技工业等重要行业和领域的，以及其他一旦遭到破坏、丧失功能或者数据泄露，可能严重危害国家安全、国计民生、公共利益的重要网络设施、信息系统等。
  2. 运营者安全保护义务：运营者应当建立健全网络安全保护制度和责任制，保障关键信息基础设施安全稳定运行，维护数据的完整性、保密性和可用性；应设置专门安全管理机构，并对机构负责人和关键岗位人员进行安全背景审查。
  3. 产品和服务安全审查：运营者采购网络产品和服务，可能影响国家安全的，应当按照国家网络安全规定通过安全审查；国家网信部门会同国务院有关部门建立网络产品和服务安全审查机制。
  4. 监测预警和应急处置：国家网信部门统筹协调有关部门建立网络安全信息共享机制，及时汇总、研判、共享、发布网络安全威胁、漏洞、事件等信息；运营者应制定网络安全事件应急预案，并定期组织演练。
  5. 监督管理体制：国家网信部门负责统筹协调关键信息基础设施安全保护工作；国务院公安部门负责指导监督关键信息基础设施安全保护工作；国务院电信主管部门和其他有关部门依照本条例和有关法律、行政法规的规定，在各自职责范围内负责关键信息基础设施安全保护和监督管理工作。
  6. 法律责任：对关键信息基础设施运营者违反本条例规定的行为，设定了责令改正、警告、罚款等处罚措施；构成犯罪的，依法追究刑事责任；给他人造成损害的，依法承担民事责任。

三、重要意义

《关键信息基础设施安全保护条例》是我国网络安全领域的又一重要立法，是落实《网络安全法》的重要举措，标志着我国关键信息基础设施安全保护进入法治化、规范化、科学化的新阶段。
